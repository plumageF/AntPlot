# Changelog

## v0.1.0 — 2026-07-15

- Added HFSS-like CSV recognition and Dataset/Curve/ReportModel workflows.
- Added S11, VSWR, realized gain, axial ratio, efficiency, HPBW, radiation-pattern and Smith-chart plotting paths.
- Added Cartesian and polar radiation-pattern rendering.
- Added multi-file and parameter-sweep curve overlays with style controls.
- Added backend-matched PNG, PDF, SVG, JSON and report export.
- Added Windows Quick Start package, public fixtures, gallery and verification archives.

Known limitations and the actual acceptance result are recorded in `docs/TEST_REPORT_v0.1.0.md`.
