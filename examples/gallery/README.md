# Example gallery / 示例图集

Each CSV in this folder has a representative image generated by AntPlot's backend. The files are intentionally small and can be used to verify a local installation.

| Input | Expected plot |
| --- | --- |
| `s11_standard.csv` | `s11_standard.png` — S11 with -10 dB threshold and working band |
| `gain_frequency.csv` | `gain_frequency.png` — gain versus frequency |
| `pattern_polar.csv` | `pattern_polar.png` — radiation pattern |
| `smith.csv` | `smith.png` — Smith chart |
| `axial_ratio.csv` | `axial_ratio.png` — axial ratio |
| `vswr.csv` | `vswr.png` — VSWR |

Use **Import data / 导入数据**, select a CSV, confirm mapping if requested, generate curves, then select the indicated plot type.
