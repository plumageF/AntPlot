# 图形处理中心 UI

React + Tailwind CSS desktop-style interface prototype for HFSS Paper Plotter.

## Run

```powershell
pnpm install
pnpm dev
```

If Node is not on PATH, use the bundled runtime paths printed by Codex workspace dependencies.

## Features

- Sidebar with plot type navigation
- Dark/light switch
- Accent color presets
- Import/export path bar
- Auto/manual setting cards
- Grid canvas with interactive network graph preview
