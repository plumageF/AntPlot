# Security Policy

## Scope

AntPlot is designed for local data processing. By default the API binds to `127.0.0.1`; do not expose it directly to the public network.

Input files are read locally and are not uploaded by the application. Do not include sensitive simulation or measurement data in public issues, pull requests or test fixtures.

## Reporting a vulnerability

Please avoid public disclosure of exploitable details. Contact the repository maintainers privately through the GitHub repository account and include reproduction steps, affected version and impact. Do not paste access tokens, private file paths or confidential RF data.

## Supported versions

The latest release receives best-effort fixes. Development snapshots may change behavior and are not production guarantees.
